## 1. Получите ID звонка
Чтобы исключить возможность отслеживания вашего аккаунта VK, просто [загуглите `"vk.com/call/join"`](https://www.google.com/search?q=%22vk.com%2Fcall%2Fjoin%22):
![Пример](https://files.catbox.moe/ic6sf9.png)

Та часть, которая идет после `https://vk.com/call/join/` - это ID звонка.

## 2. Решите капчу
Мы делаем всё возможное, чтобы автоматическое решение капчи работало корректно, но если на данный момент оно не работает, выполните следующее:

### 2.1. Установите юзерскрипт
Когда все попытки автоматического решения капчи исчерпаны, вы увидите что-то похожее на это:
```
2026-04-22 16:09:19.748 [INFO] manual captcha solve required userscript=http://localhost:1984/vk_manual_captcha.user.js guide=http://localhost:1984/ url=https://vk.com/call/join/... timeout=10m0s
```

Сначала установите [TamperMonkey](https://addons.mozilla.org/ru/firefox/addon/tampermonkey/). Ссылка на страницу расширения для Firefox указана намеренно - Firefox позволяет устанавливать и использовать расширения даже на мобильных платформах, например на Android. Обязательно разрешите расширению работать в режиме инкогнито.
![Tampermonkey](https://files.catbox.moe/y5vlyg.png)

Затем скопируйте URL из поля `userscript` в строке лога и откройте его в браузере. Вы увидите примерно следующее:
![Установка](https://files.catbox.moe/pq98f9.png)

Нажмите кнопку установки.

### 2.2. Пройдите капчу
Теперь перейдите по URL из поля `url` и вручную пройдите капчу. Юзерскрипт автоматически нажмёт кнопку "Присоединиться" и перенаправит вас с VK после успешного получения всех необходимых токенов. Если вы авторизованы в VK, используйте режим инкогнито.
